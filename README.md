# Telegram Captcha Invite Bot

Бот выдаёт временную ссылку в группу после капчи.

## Установка
1. Клонируйте: `git clone <repo>`
2. `cp .env.example .env` → заполните TOKEN, GROUP_ID
3. `pip install -r requirements.txt`
4. `python bot.py`

## Деплой 24/7
- Bothost.ru (free)
- Render.com
